PHP 程式檢測工具

1. php -l xxx.php
 最簡單，僅檢查 syntax
2. [PHPSA](https://github.com/ovr/phpsa)
 phar或composer安裝，靜態檢查工具
3. [PHP_CodeSniffer](http://pear.php.net/package/PHP_CodeSniffer/)
 phpcs，曾經存在過
4. [PHPMD](https://phpmd.org)
 一樣 phar或composer安裝，仿自 java 的 PMD
