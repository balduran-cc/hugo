Google Chart API

Published Tue, Dec 11, 2007 by Balduran Chang

使用 Google Chart API 可以動態地生成圖表，只要使用 http://chart.apis.google.com/chart?<parameter 1>& 這種形式的 URL 就可以產生圖表，不用申請 API，沒有繁雜的編寫過程，只要收集完整的 dataset，圖表隨手可得。

wordpress 有個 SparkStats Plugin，可以產生名為 Sparkline 的線圖，我想這個 API 讓這個功能更簡單實做了吧。
