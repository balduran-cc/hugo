最受歡迎的 twitter client

Published Wed, Nov 21, 2007 by Balduran Chang

直接照著 The RWW Guide to the World’s Most Popular Twitter Clients 寫，裡面有些我有用過，部分則無，而我不是重度使用者，並不傳送訊息到手機上。 twitter 現在有多重要呢？以 blogblind 為例，就是 Wraecca 先丟上 twitter，引起眾人回應，之後的聊天話題不斷。

twitter publish tool

  1. web
  2. twitterrific
  3. IM
  4. txt

web 是直接從 twitter 頁面發表，很直覺也最多人使用。

twitterrific 是 MAC 上的軟體，用過，確實不錯。

IM 用即時通訊軟體傳送訊息到 twitter@twitter.com。

txt 則是簡訊，要收費，應該不普及吧。

其他的就不看了，使用者太少就忽略它。
