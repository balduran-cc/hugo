board game on iPhone - 2

Published Thu, Jun 25, 2015 by Balduran Chang

繼之前 board game on iphone，再列出一些好玩的 board game App

ascension

中文名字是暗殺神，卡牌遊戲，遊戲內可以再購買牌組。

blokus

益智遊戲，乍看之下很像俄羅斯方塊，目的是要卡地盤，規則不難但要贏也不簡單

New World Colony

策略型遊戲，很像沒有骰子的卡坦島。

hive online

昆蟲棋，蜜蜂棋，堵死對方的蜂后就贏，一開始非常讓人搞不懂。網友介紹的比較詳細的文章
