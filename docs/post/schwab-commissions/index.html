嘉信調降交易手續費

Published Wed, Feb 15, 2017 by Balduran Chang

Schwab 的美股交易手續費降為 6.95 美金，先前為 8.95 美金。現在開戶比較難了，資金需求低標是兩萬五美金。

TD 還是維持 9.99 美金的手續費。

Firstrade 則是從 6.95 調降為 4.95，原本 Schwab 調降是往 Firstrade 靠攏，現在又拉開了差距。

Schwab 還有一大利基是提供國際客戶 Visa Debit card，這一優勢目前無人可取代。
