Google Browser Sync is no longer available for download

Published Fri, Aug 22, 2008 by Balduran Chang

Google Browser Sync 是給 Firefox用的 extension，可以將 FF 的設定、書籤、cookies、密碼在不同電腦間做同步，使用者在使用時所開啟的session頁面都會儲存。

不過現在這個服務不再提供下載了，因為 FF 進到三版以後，mozilla 自己有自行規劃的同步軟體weave，還有其他的 extension 也是同樣功能，google 自己也將它整合進 toolbar 裡，因此 google 不再提供更新，但還是開放程式碼供後進發展新 project。
