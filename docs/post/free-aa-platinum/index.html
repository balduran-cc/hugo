免費 AA 白金會籍

Published Tue, Sep 26, 2017 by Balduran Chang

AA 美國航空目前提供免費的白金會籍，不過並不是註冊就有，被選中的顧客會得到獨特的 promote code，並且可以升為白金三個月，要維持白金資格的話，要飛 12500 EQM 和 2000 EQD。

這應該算是發掘潛在客戶，提高貢獻度吧。
