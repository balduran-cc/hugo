Project Early 一首情歌給我

Published Tue, Sep 23, 2008 by Balduran Chang

Project Early 是由奇哥和馬念先所組成。奇哥，自然捲的靈魂人物。馬念先，糯米團主唱，海角七號的馬拉桑，這首「一首情歌給我」我很喜歡。

因為掀開後車箱注意到奇哥，因為海角七號，重新認識了馬念先，也開始懷念起糯米糰的「跆拳道」、「巴黎草莓」，也才得知這個兩人合組的團體，說到底，海角七號每個角色的背後，都代表著一段音樂時期呢。
