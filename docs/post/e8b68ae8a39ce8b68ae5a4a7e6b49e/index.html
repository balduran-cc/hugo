越補越大洞

Published Thu, Feb 12, 2004 by Balduran Chang

請看M$的網站。
Microsoft 安全性公告 MS04-007
發布的時間是2月11號，其中提到

如果未安裝 MS03-041 Windows NT 4.0 安全性更新，就不需要安裝此安全性更新。

再往這MS 03-041一看，發布時間是2003年10月中，同樣的也是列為重大更新。更新了舊的漏洞，出現了新的，而且還是在五個月後才提出解決方法，這不是越補越大洞嗎?
當然微軟的作業系統漏洞一堆也不是一天兩天的事了，不過一直需要注意update也實在是很累人，包括從以前到現在的著名病毒，矛頭全都指向win平台，使用電腦，還要時時注意是否中毒中木馬，當然這只是個小牢騷，世界上不會有完全完美的作業系統，防毒軟體也會有吃鱉的時候，只是，在資訊產品充斥的現在，資訊安全實在令人擔心。
