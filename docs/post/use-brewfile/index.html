使用 homebrew 快速建立環境

Published Mon, Dec 11, 2017 by Balduran Chang

使用 homebrew 至少也有四五年，真的是 mac 上套件管理的好工具，現在連應用程式都可以幫你裝，大幅減少因為換機而需要建立環境的時間。

現在學到一個新招是使用 Brewfile 一次大量安裝自己需要的套件，概念就跟 npm install 一樣。

這是我自己的 Brewfile，只要安裝好 homebrew，再輸入brew bundle install就會依序安裝。

其中我也使用了 mas-cli，用來安裝 Mac App store上的程式。
