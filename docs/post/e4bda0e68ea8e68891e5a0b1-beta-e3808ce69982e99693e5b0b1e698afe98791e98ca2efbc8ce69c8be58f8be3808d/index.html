你推我報 beta - 「時間就是金錢，朋友」

Published Wed, Sep 24, 2008 by Balduran Chang

既然你推我報在這裡留下了留言，那就來看他們在幹嘛。

你推我報分成兩部分，你推和我報，你推是 digg-like 的推文互動，我報是 vlog 形式的對應內容。

分開來看，你推的部分可以不用著墨太多，這類型網站不是第一次出現，能不能成功要看的是使用者人數，從熱門第一篇的推文人數是個位數來看，可以直接下定論，使用者不多，所以才要用額外的方法增加曝光率。

針對較為「熱門」的話題，由工作人員製作數位內容，呈現一個主題，有點像是個主題節目一般，就是我報，想借由影音吸引目光，但是！我覺得這些內容一點都不好看，講難聽一點，浪費我六分鐘的時間得到一個無言體驗。

time is money, friend.
