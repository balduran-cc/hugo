Windows Live Calendar 正式上線

Published Sun, Nov 11, 2007 by Balduran Chang

原本還在 betas 的日曆已經正式轉為公開上線了。可以從 http://calendar.live.com/calendar/ 用MSN帳號登入使用。

幾個特點:

  • 沒有繁體中文。
  • 特別的Social Event。
  • 直接在日期上新增事件。
  • 支援多個日曆。
  • 分享設定有 Html, iCal, Xml。

其中在操作上的拖拉點選實在和 google calendar 很相像，也是使用了大量的 ajax。
