Oracle and Sun

Published Thu, Apr 23, 2009 by Balduran Chang

滿出乎意料的，甲骨文買下了昇陽，並不是謠傳的 IBM。我知道 Oracle 是大公司，可是還真不知道可以吃下另一家，起碼我心中昇陽還不小。

Oracle 最有名的是資料庫，SUN 最有名的是 JAVA，兩個的交集合是 JDBC(冷)。在MySQL 被 SUN 買下之後，Oracle 又吃了 SUN，這是不是說 MySQL 前途堪憂啊(搖手)？

額外一提，Google App Engine 支援 JAVA 了。
