TD Ameritrade 調降交易手續費

Published Fri, Mar 3, 2017 by Balduran Chang

繼 Schwab 和 Firstrade 降交易手續費之後，TD Ameritrade 也跟進 了，調降到 6.95 美金，和 Schwab 打平。

目前台灣人可以開的美國券商基本上就是這三家，Etrade 已經不再提供國際客戶開戶，TD Ameritrade 有免佣 ETF，schwab 有 Visa debit card，firstrade 有中文服務支援。
