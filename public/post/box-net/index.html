box.net 檔案分享

Published Thu, Apr 17, 2008 by Balduran Chang

由 googoglopoly 知道的服務，當然是先由 google 抓到我的注意力，後來才注意到 box.net。 在 web2.0 的服務當中我不常用的就是免費檔案傳輸，因為不太會將檔案放在網路上供人下載，但是我對這 box.net 還是試用了一下。 我覺得很新鮮的是它的上傳方式，一般網頁提供的選擇檔案之後上傳這當然有，還另外提供了 java Applet讓使用者可以拖拉放 (drag & drop)的方式上傳檔案，這很方便好用，重點免安裝。

順道找了其他的 file hosting 服務，BADONGO無拖拉放，SkyDrive無拖拉放，yousentit無拖拉放，rapidshare無拖拉放。大部分提供的額外上傳機制都是要額外安裝軟體、activeX。

還看到中華電信safebox，居然不能 share，那開來幹嘛？
OpenBox Service 可以讓其他線上服務存取你的檔案，就真的有線上硬碟的感覺了。FTP 已經是歷史了，傳檔案還是用新服務吧。
