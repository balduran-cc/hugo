Hilton HHonors update

Published Mon, Feb 13, 2017 by Balduran Chang

Hilton 的常客計畫 HHonors 在二月初的時候發佈了一些更動，整體來說是改得更好了。

  1. HHonors 改為 Honors 沒錯，就是少了個 H 而已
  2. 彈性的 Cash + Point 目前的點數房是按照飯店等級來劃分的，總共有十級，之後兌換可以自由選擇現金和點數的比例，彈性更高，零碎的點數也不會浪費。
  1. 全家桶 最多允許十個親友的點數可以跟自己合併
  2. 在 Amazon上使用 可以在 Amazon上用點數折抵消費，通常是不划算的，可以忽略。

另外，在 2017年年底前入住 Conrad 或 Waldorf Astoria，每晚可以多得 2000點，請在此註冊，訂房時要輸入 code PBK2P1。
