flickr 推出 Collections 功能

Published Wed, Mar 14, 2007 by Balduran Chang

FlickrBlog 解釋了新功能 collection，即為 “sets of sets”，不再是以前只有單一sets能管理你的相簿了，現在可以將多個 sets 放在同一個 collection，而 set 不必存在唯一一個 collection，這是很有彈性的組織方法。

這項功能只有 pro 付費會員能使用，可改變相簿 layout，選擇用 set 還是 collection。
