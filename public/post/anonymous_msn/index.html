msn 可從網頁接收匿名訊息

Published Thu, Nov 15, 2007 by Balduran Chang

live messenger 現在推出一個 web widget，允許網友可以從 網頁上的 widget 傳送訊息到 live messenger 上，可以選擇用自己的帳號或是匿名，而 widget 上會顯得是你的暱稱，因此不用擔心帳號被有心人得知。

想要取得這個 widget 不需填入任何訊息，登入 windows live ID 之後，複製他提供的程式碼就可以了。你也可以選擇只要顯示 msn 狀態，這樣比較有隱私一點。
